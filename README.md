## Getting Started (Development)

1. **Clone the repository:**
   ```sh
   git clone https://github.com/Mansi-Narang/Desky.git
   cd Desky
   ```

2. **Install dependencies:**
   ```sh
   npm install
   ```

3. **Start the app:**
   ```sh
   npm start
   ```
